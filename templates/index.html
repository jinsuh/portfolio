Countdown Android Application

Based on the game show "Countdown". Arrange five numbers with the right operations to reach the target number! Published to the Google Play Store.

Easy Money First Prototype

This is my current ongoing Unity project. Web player needs Firefox, Safari, or Internet Explorer, but the game can be downloaded here. An XBOX controller is required as this is a local coop game. The game's first prototype was targeted as a co-op stealth heist game. Art and the guard AI was written by a friend while the mechanics and gameplay was written by me. Some art and animation was added into the game and a trailer was made to promote our game. I learned Adobe Photoshop and Premiere to video edit the trailer. The game was written in C# on the Unity platform.

The current state of the game has changed drastically. We've added a graphics artist to our team and have started to move to 3d models. We're also adding a new core mechanic to the game. The theme is a psychological puzzle game. We're super excited and are scheduled to have a demo out in late August!

Maze Runner

A Unity game project. Web player still needs Firefox, Safari, or Internet Explorer but the game can be downloaded on the same Google Drive. Controls are WASD to move and QE to rotate. The game procedurally generates a maze and the goal is to find the "food" (a cube) that can be seen on the minimap, before your hunger score reaches 0. Only the room you are in is shown on the minimap. The game was written in C# on the Unity platform.

Emotipic

Check out the devpost! This is an Android application that uses Microsoft's Emotion API to detect the emotion of a selfie and adds extra touches to the image. Android application was developed in Java with Android Studio while web end was written in HTML, CSS, and Javascript along with Node.js.

Local Business Search Android Application

An Android application that first uses the Google Location Services to get GPS coordinates and then uses the Yelp API to search for businesses nearby. APK can be found on the Google Drive. Also uses a SQL database to store searches to be able to view the list while offline. Written in Java on the Android Studio platform.

Khan Academy Badge Viewing Android Application

An Android application that first uses the Volley to retrieve Khan Academy Badge Information. Wrote an image cache class to handle the multitude of images. APK can be found on the Google Drive. Written in Java on the Android Studio platform.

Tic tac toe

Node project that uses express and socket.IO. This requires two windows on this website to start playing. Additional people can open the site and spectate the match. Click reset and refresh for the next two people to play. Written in HTML, CSS, and JavaScript.

Monty Hall Problem

A visualization and explanation of the Monty Hall Problem written with Khan Academy's programming platform. Allows users to manual play the game or run a simulation of the problem and record the statistics with a graph. Written in JavaScript.
